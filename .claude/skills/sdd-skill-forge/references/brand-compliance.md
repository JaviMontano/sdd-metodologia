## Visual Rules

- Swiss 8px grid for spacing [EXPLICIT]
- Navy-tinted soft shadows [EXPLICIT]
- Cards: glassmorphism `blur(16px) saturate(180%)` + `rgba(255,255,255,0.03)` [EXPLICIT]
- Flat vector illustrations: faceless figures, soft geometric shapes [EXPLICIT]
- High legibility: large text, high contrast, never on noisy backgrounds [EXPLICIT]
