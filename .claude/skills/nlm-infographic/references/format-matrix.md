# NLM Infographic — Format Matrix

> 10 combinaciones optimizadas de orientación × estilo × nivel de detalle.

| ID | Caso de Uso | orientation | detail_level | infographic_style | Platform Target |
|----|-------------|-----------|-------------|-------------------|----------------|
| IM-01 | Executive dashboard | landscape | standard | professional | Presentación, email |
| IM-02 | Instagram story / reel cover | portrait | concise | editorial | Instagram, social |
| IM-03 | Social media post | square | concise | bento_grid | LinkedIn, Twitter |
| IM-04 | Technical reference | portrait | detailed | scientific | Documentación, wiki |
| IM-05 | Tutorial paso a paso | portrait | standard | instructional | Blog, onboarding |
| IM-06 | Brainstorm / concept map | landscape | standard | sketch_note | Workshop, Miro |
| IM-07 | Blog header / banner | landscape | concise | editorial | Blog, newsletter |
| IM-08 | Data comparison table | landscape | detailed | professional | Reporte, deck |
| IM-09 | Engaging / friendly overview | square | standard | kawaii | Redes, onboarding |
| IM-10 | Architecture diagram | landscape | detailed | bento_grid | Técnico, documentación |

## Cuándo usar cada combinación

### IM-01: Executive Dashboard
- **Audiencia**: C-level, stakeholders
- **Contenido**: KPIs, métricas, decisiones clave
- **NO usar si**: el contenido es narrativo o conceptual

### IM-02: Instagram Story
- **Audiencia**: Público general, followers
- **Contenido**: 1 dato impactante + visual llamativo
- **NO usar si**: necesitas mostrar datos complejos

### IM-03: Social Media Post
- **Audiencia**: LinkedIn, Twitter, comunidad profesional
- **Contenido**: Comparación rápida, checklist, tip visual
- **NO usar si**: es contenido técnico profundo

### IM-04: Technical Reference
- **Audiencia**: Ingenieros, investigadores, técnicos
- **Contenido**: Diagrama detallado, especificaciones, flujos
- **NO usar si**: la audiencia no es técnica

### IM-05: Tutorial Paso a Paso
- **Audiencia**: Principiantes, estudiantes
- **Contenido**: Proceso secuencial (5-8 pasos)
- **NO usar si**: no hay proceso lineal

### IM-06: Brainstorm / Concept Map
- **Audiencia**: Equipo de trabajo, facilitadores
- **Contenido**: Ideas interconectadas, relaciones, clusters
- **NO usar si**: los datos son cuantitativos

### IM-07: Blog Header
- **Audiencia**: Lectores del blog, newsletter
- **Contenido**: Título visual + 1-3 datos clave
- **NO usar si**: necesitas profundidad

### IM-08: Data Comparison
- **Audiencia**: Analistas, decisores
- **Contenido**: Tabla comparativa visual, pros/cons, rankings
- **NO usar si**: hay menos de 3 elementos a comparar

### IM-09: Friendly Overview
- **Audiencia**: General, no técnica, youth
- **Contenido**: Resumen amigable, conceptos principales con iconos
- **NO usar si**: el contexto es formal/corporativo

### IM-10: Architecture Diagram
- **Audiencia**: Arquitectos, tech leads
- **Contenido**: Componentes, conexiones, flujos de datos
- **NO usar si**: no hay estructura modular
